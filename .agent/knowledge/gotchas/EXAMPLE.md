# Example Gotcha — Delete Me

> Template shape for a gotcha file. Delete this file once a real gotcha is documented.

- **Symptom:** What you see — the error message, the unexpected behaviour, the test failure.
- **Cause:** Why it happens — the underlying mechanism, the library quirk, the config default.
- **Solution:** How to fix or avoid it — concrete steps or code snippet.
- **Discovered in:** Project name, batch number (if applicable), date (YYYY-MM-DD).
- **Affects:** Versions, conditions, environments where this reproduces.
